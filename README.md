## For `MATCHES` it allows to:
- Read all
- Create
- Edit
- Delete (physical delete)

## For `USERS` it allows to:
- Read all
- Log In
- Create
- Update
- Delete (logic delete)
- Activate

## Technologies
- Node JS
- Express
- Mongo DB
- Mongoose

## You can install it using the following commands
- Clone the repository
- Run `npm install`
- Run `npm run dev` (it will run in dev mode using nodemon)
